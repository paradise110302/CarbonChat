# Backend

This is our backend app deployed to heroku

## Local Dev

### Build
1. Install docker
1. Navigate to the source of the repo
1. Run : `script/build-web.sh`

### Run it locally

1. First build it, open a shell and navigate to the source of the repo
1. Run : `./script/dev-run.sh`
